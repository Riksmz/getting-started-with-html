So, how does the internet work, generally?

The internet is connected through various hosting servers that allow users to link up with each other, and incredibly, share information. This information usually comes in the form of web pages, which we access through web browsers. There doesn't seem to be a limit to the amount and type of information that can be shared amongst its users, so long as you have the know-how and proper resources.

What exactly is HTML?

HTML is exactly hypertext markup language. HTML is the text content of the web page, and the links and references to other documents contained therein. Basically, its everything you'd see when you open the web page.

Tags and elements:

Elements are the building blocks of the HTML language that are used to display the information on the web page. Now, in order to tell our computer which elements to use an how to use them, we need tags. Tags are essentially our instructions to the program.

Amongst these tags, we have some that are inline elements and some that are block elements. Inline elemnts can be inserted into lines of content without causing a break, whereas block elements will actually create a sort of "invisible box" around the content, which may even seperate it to itself.
